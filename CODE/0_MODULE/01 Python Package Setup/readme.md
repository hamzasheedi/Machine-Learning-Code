
# Step 2: Python Package Setup  
## 3. Pip Package Installer  
- Pip comes bundled with Python 3.13.  
- It is used to install and manage Python libraries. 
- Verify pip installation by running the following command in the terminal: 

```  python
pip --version 
``` 

4\. Common Python Libraries for AI/ML
-------------------------------------

Install essential Python libraries for Artificial Intelligence and Machine Learning using pip.

### Required Libraries

*   **numpy** – numerical operations
    
*   **pandas** – data handling and analysis
    
*   **matplotlib** – data visualization
    
*   **seaborn** – statistical data visualization
    
*   **scikit-learn** – basic machine learning algorithms
    
*   **tensorflow / pytorch** – deep learning frameworks
    

### Example Installation Command

```
pip install numpy pandas matplotlib seaborn scikit-learn tensorflow   
```

**Note:**
Make sure you have an active internet connection before installing packages.
